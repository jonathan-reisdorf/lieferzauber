Lieferzauber.de

Lieferzauber.de ist kein geschäftsmäßiger Dienst, sondern ein privates, komplett nichtkommerzielles Projekt und unterliegt daher auch nicht der Impressumspflicht nach §5 TMG!

Dennoch vermittelt Lieferzauber.de per API unentgeltlich Leistungen der Lieferheld GmbH, daher gelten im Fall einer Bestellung für Verbraucher die AGB sowie Widerrufsrecht, Datenschutzbestimmungen und das Impressum der Lieferheld GmbH.

Wir distanzieren uns audrücklich von diesen Leistungen und erklären hiermit, in keinerlei Verbindung mit der Lieferheld GmbH zu stehen. Diese Website stellt lediglich ein Werkzeug dar, dessen Benutzung komplett auf eigene Verantwortung und Gefahr geschieht. Wir übernehmen keinerlei Haftung.

Ebenso nutzt diese Website zur Addressvalidierung die Google Maps Geocoding API.

Datenschutz
Lieferzauber.de speichert serverseitig keinerlei Daten ab. Die Speicherung der Daten findet komplett im Browser des Nutzers statt, wo diese auch bis zur Löschung durch diesen verbleiben. Es werden von Lieferzauber.de keine Daten ausgewertet. Die Addressdaten werden allerdings zur Validierung an die Google Maps Geocoding API gesendet, wo sie eine anonyme Addressanfrage darstellen die nur aus Straße, Hausnummer, Postleitzahl, Stadt und Land besteht (es werden also keine persönlichen Details übersendet).
Im Fall einer Bestellung werden die Lieferadresse sowie die Informationen über die Bestellung an die Lieferheld API übersendet und ein Gastkonto angelegt. Dies sind weniger Daten als die Lieferheld GmbH zur Erbringung des Dienstes auf ihrer Seite selbst erheben würde.
